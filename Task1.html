Tic Tac Toe Game

The Python implementation of Tic Tac Toe revolves around a 3x3 grid where players take turns marking 'X' or 'O'. The game checks after each move for a winning combination across rows, columns, and diagonals. Players input their moves via a command-line interface, and the game continues until one player wins or the board fills up, resulting in a draw.
